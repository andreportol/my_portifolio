Logo abaixo segue uma demostração de alguns dos projetos que foram elaborados por mim. A grande maioria desses projetos estão no repositório do GitHub para download.

Este projeto foi desenvolvido para Coordenadoria de Gestão de Áreas Públicas-CGAP do município de Campo Grande-MS. Consiste em controlar o andamento da documentação, os prazos das notificações e dos autos de infração gerados após vistoria no local. Com esse aplicativa, é possível cadastrar todos os infratores e controlar os laudos de vistoria elaborados pelos auditores fiscais de meio ambiente.
